

#### kibana 使用

当kibana链接上ES之后, 打开Discover会看到对应的数据集,
打开之后在左下角可以进行数据字段的展示配置, 一般使用 message 和 fields.type 两个字段就行

检索使用KQL语言 使用方法

例如在message字段中检索信息,

message:"XNIO-1 task-8"

就会检索message中包含'XNIO-1 task-8'的数据

PUT

{

}
